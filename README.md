# 🎵 ハロ／ハワユ - 일본어 가사 학습 앱

음악과 함께하는 일본어 학습 웹 애플리케이션

## ✨ 주요 기능

- 🎵 **오디오 싱크**: 음악 재생과 완벽하게 동기화된 가사 하이라이트
- 📝 **4단 표시**: 일본어 원문 / 후리가나 / 로마자 / 한국어 해석
- ⭐ **북마크**: 학습하고 싶은 가사 저장
- ✅ **아는 단어 표시**: 이미 아는 가사 체크
- 🎯 **가사 클릭 점프**: 가사 클릭 시 해당 부분으로 음악 이동
- 📊 **학습 통계**: 북마크, 아는 단어, 진행률 추적

## 🚀 사용 방법

1. 웹사이트 접속
2. "오디오 파일 선택" 버튼 클릭
3. **ハロ／ハワユ** 음원 파일 업로드 (MP3/M4A 등)
4. ▶️ 재생 버튼 클릭
5. 가사가 음악에 맞춰 자동으로 하이라이트됨

## 📱 모바일 지원

- ✅ iOS (iPhone/iPad)
- ✅ Android
- ✅ 반응형 디자인
- ✅ 터치 인터랙션

## 🎨 기능 설명

### 북마크 (⭐/☆)
학습하고 싶은 가사를 북마크하세요. 브라우저를 닫아도 저장됩니다.

### 아는 단어 (✅/⭕)
이미 아는 가사를 체크하세요. "아는 단어 제외" 토글로 숨길 수 있습니다.

### 가사 클릭
가사를 클릭하면 해당 부분부터 음악이 재생됩니다. 반복 학습에 유용합니다.
